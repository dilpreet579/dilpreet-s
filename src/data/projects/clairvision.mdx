# ClairVision

Last Updated - 02 March 2026

Built an AI-powered computer vision pipeline for event photo curation, automating duplicate detection and blur filtering.

## What it Does
- Applied **CLIP ViT-L/14, FaceNet, MTCNN, and ResNet-50** for embedding generation, face detection, and quality scoring, achieving 97% clustering precision and 98% blur detection accuracy.
- Optimized large-scale similarity search using **FAISS**, enabling sub-second retrieval and clustering on datasets exceeding 10k images.

## Technology Stack
- Python
- PyTorch
- FAISS
- ResNet-50 / FaceNet
- Vision Transformers (ViT)
