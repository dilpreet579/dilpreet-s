# Real-Time Bot Detection System

Built a real-time ML system to detect and block malicious bot purchases during high-demand holiday sales for the Walmart Sparkathon 2025.

## What it Does
- Trained a statistical ML model on large-scale user behavior data, achieving **99% detection accuracy** in simulations.
- Reduced false positive rates by 40% through advanced statistical modeling and hyperparameter optimization.

## Technology Stack
- Python / FastAPI
- Next.js
- Streamlit
- Selenium / Puppeteer / Playwright
